# MeetingMind Privacy Policy

**Effective Date:** December 2024

## Data Collection
MeetingMind processes meeting notes you provide to extract actionable tasks. We collect:
- Meeting transcripts you input
- Jira user information for task assignment
- Basic usage analytics (number of meetings processed, tasks created)

## Data Usage
- Meeting notes are processed by AI services (OpenAI) to extract tasks
- User data is used only for task assignment within your Jira instance
- Analytics help improve app performance

## Data Storage
- Meeting notes are not permanently stored
- Analytics data is stored securely within Atlassian Forge infrastructure
- No personal data is shared with third parties except OpenAI for processing

## Data Security
- All data transmission is encrypted
- App runs on Atlassian Forge secure infrastructure
- OpenAI API calls use secure HTTPS connections

## Your Rights
- You control all data input into the app
- Meeting notes are processed temporarily and not retained
- You can stop using the app at any time
